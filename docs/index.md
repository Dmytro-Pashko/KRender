# KRender SDK

KRender SDK is a Kotlin and libGDX engine workspace built around a backend-neutral core, a LibGDX runtime backend, a scene player, and standalone editor tools.

This site publishes the repository documentation through GitHub Pages.

## Documentation Areas

- `Architecture` covers the engine runtime, backend boundary, rendering pipeline, assets, UI, scene player, and logging.
- `Tools` documents the editor tools for assets, models, animations, terrain, scenes, and UI composition.
- `Quality` collects notes about static analysis and test coverage.
